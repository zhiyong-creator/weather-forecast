# WeatherWidget — 天气预报微信小程序

基于和风天气 v7 API 的微信小程序，提供实时天气、24小时逐时预报、3天预报、生活指数等全面气象信息服务，支持自动定位与手动城市搜索。

## 技术栈

| 类别 | 技术 |
|------|------|
| 框架 | 微信小程序原生框架 |
| API | 和风天气 v7（实时天气 / 24h预报 / 3天预报 / 生活指数 / 城市查询） |
| 数据流 | Promise 链式调用 + 事件总线（EventEmitter） |
| 样式 | WXSS 原生样式，毛玻璃卡片 + 渐变背景 + 文字投影可读性增强 |
| 存储 | wx.Storage 本地缓存（设置项、天气快照、用户信息） |

## 页面功能详解

### 登录页 (pages/login)

- 一键微信登录，调用 `wx.login` 获取 code
- 登录成功自动跳转首页，失败提示重试
- 按钮防重复点击（`canLogin` 锁）

### 首页 (pages/index)

| 模块 | 说明 |
|------|------|
| 搜索栏 | 城市名搜索，支持中文/拼音；半透明背景 + text-shadow |
| 顶栏 | 用户头像 + 昵称 + "更新于 MM-dd hh:mm" |
| 当前天气 | 城市名（粗体 + 阴影）+ 大字号温度 + 天气状况文字 |
| 逐时预报 | 水平滚动卡片列，依次显示时间 → 温度 → 天气 → 风力 |
| 3天预报 | 三列毛玻璃卡片（今天/明天/后天）+ 温度渐变色条 + 天气 + 风力 |
| 生活指数 | 水平滚动，图标 + 名称 + 金色等级 + 描述文字（运动/洗车/穿衣/紫外线/旅游/舒适度/感冒） |
| 实时详情网格 | 12 项指标：温度/体感/湿度/降水量/风向/角度/风力/风速/能见度/气压/云量 |
| 背景图 | 7 张风景图，根据天气自动匹配；支持手动切换 |

**生命周期行为：**

- `onLoad` — 缓存快照渲染 → 事件总线注册 → 发起定位 → 并发请求 API
- `onShow` — 刷新用户信息，跳过分享回退场景
- `onPullDownRefresh` — 带 `_loading` 防重入锁，重新定位加载
- `onShareAppMessage` — 分享卡片标题 + 封面图

**数据加载流程：**

```
getLocation() → lookupCity(坐标) → 反查中文城市名
  → Promise.all([getWeatherNow, getWeather24h, getWeather3d, getIndices])
  → parse*() 解析 → setData() 渲染
```

单个接口失败不影响其余模块（`.catch(() => null)` 容错）。

### 城市选择页 (pages/citychoose)

- 城市列表按 A-Z 字母分组排序
- 搜索过滤：城市名包含输入文字即命中
- "猜你想找" 区域：热门城市 + 定位入口
- 选中后通过 `events.emit('weatherRefresh', name)` 通知首页刷新

### 设置页 (pages/setting)

| 功能 | 说明 |
|------|------|
| 隐藏搜索栏 | 开关，首页搜索栏显隐 |
| 隐藏生活指数 | 开关，首页生活指数模块显隐 |
| 屏幕常亮 | `wx.setKeepScreenOn` 开关 |
| 屏幕亮度 | 滑块调节，实时生效 |
| 强制更新 | 检测小程序新版本并提示重启 |
| NFC 检测 | `wx.getHCEState` 检测设备 NFC 支持 |
| 用户信息 | 头像选择（`open-type="chooseAvatar"`）+ 昵称输入 |
| 退出登录 | 清除缓存 → 跳转登录页 |
| 初始化设置 | 单项清除设置缓存 |
| 清除全部数据 | `wx.clearStorage` 所有数据 |
| 系统信息 | 跳转 systeminfo 页 |

### 系统信息页 (pages/systeminfo)

- 展示 14 项设备/系统参数：品牌、型号、像素比、屏幕尺寸、语言、微信版本、操作系统、基础库版本等

## 关键逻辑模块

### services/api.js — 请求层

- `request(url, data)` — 通用 Promise 封装，校验 `code === '200'`
- `getWeatherNow` / `getWeather24h` / `getWeather3d` / `lookupCity` / `getIndices`

### services/weather.js — 解析层

- `parseNowWeather(nowData, fxLink, cityName)` — 优先使用传入中文城市名，兜底从 fxLink 提取拼音
- `parseHourlyWeather(hourlyList)` — fxTime 截取 HH:mm
- `parseDailyForecast(dailyList)` — 自动生成"今天/明天/后天"日标签
- `parseIndices(indicesList)` — 7 类指数图标映射（运动/洗车/穿衣/紫外线/旅游/舒适度/感冒）
- `getBackgroundByWeather(weatherText)` — 按晴/雨/雪/云/雾/阴关键词匹配背景 + 顶栏色

### utils/events.js — 事件总线

- 跨页面通信：城市选择 → 首页刷新（`weatherRefresh`）
- 设置变更 → 首页重载（`settingChanged`）

### components/heartbeat — 彩蛋

- 搜索框输入 `520` 或 `521` 触发爱心飘落动画
- 随机数量 + 随机轨迹 + 旋转，5 秒自动结束

## 样式设计要点

- 全局底色 `#f4f6f9`，内容区铺满天气背景图
- 数据模块叠加半透明黑色 `rgba(0,0,0,.4~.6)`，层次分明
- 标题栏：26rpx 加粗 + 字间距 + 下划线分隔
- 详情网格：标签 22rpx 半透明白，数值 26rpx 加粗纯白
- 所有主要文字带 `text-shadow` 投影，在浅色背景图上保持可读性
